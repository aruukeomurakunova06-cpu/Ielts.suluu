# IELTS Suluu (ИELTS Сулуу) 🇰🇬✨

**IELTS Suluu** — это красивая, бесплатная и эффективная веб-платформа для интерактивной подготовки к экзамену IELTS. Название отражает философию проекта: подготовка к сложному академическому экзамену может быть эстетичной, понятной и увлекательной.

[🌐 Открыть сайт в браузере (Live Demo)](https://ваше-имя-пользователя.github.io/ielts-suluu/) *(замените на вашу ссылку после включения GitHub Pages)*

---

## ✨ Ключевые особенности
* **Красивый дизайн (Suluu UI):** Премиальный темный интерфейс на базе Tailwind CSS, создающий фокус на обучении и снижающий нагрузку на зрение.
* **4 модуля для прокачки Skills:** Комплексные интерактивные блоки для тренировки Reading, Listening, Writing и Grammar.
* **ИИ-Помощник:** Персональный виртуальный тьютор, готовый в любую секунду подсказать структуру эссе, грамматические правила или лайфхаки для секции Reading.
* **Игровой прогресс (XP Счёт):** За каждое выполненное задание пользователь мгновенно получает очки опыта, отслеживая свой локальный результат.

## 🛠️ Технологический стек
* **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS v4.
* **Иконки:** Font Awesome 6.
* **Хостинг:** GitHub Pages (100% бесплатно и без серверов).

## 🚀 Как развернуть проект у себя
1. Склонируйте репозиторий: `git clone https://github.com/ваше-имя/ielts-suluu.git`
2. Откройте файл `index.html` в любом браузере. Бэкенд-сервер не требуется!
